# clone-wikipedia
Estrutura do Projeto
Plaintext
├── index.html            # Arquivo HTML principal (página do Palmeiras)
├── style.css             # Arquivo de estilização (layout, cores e responsividade)
├── arquivo.js            # Lógica em JavaScript (sistema de busca e transição de páginas)
└── estadiopalmeiras.jpeg # Imagem do estádio Allianz Parque
🛠️ Tecnologias Utilizadas
HTML5: Estruturação de texto, listas, imagens e links.

CSS3: Estilização (cor temática #00590d aplicada ao cabeçalho alviverde).

JavaScript (ES6): Manipulação de eventos via função carregarTime() para tornar a enciclopédia dinâmica.

⚙️ Como Executar o Projeto
Faça o clone ou baixe os arquivos deste repositório.

Certifique-se de que os arquivos style.css e arquivo.js estejam na mesma pasta que o index.html.

Abra o arquivo index.html em qualquer navegador web moderno (Chrome, Edge, Firefox, Safari).
